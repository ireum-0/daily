# Daily 기능 메모

## 목적

Daily는 NEIS에서 학교 급식 정보를 가져와 보여주고, 사용자가 데이터 출처, 학교, 선호 메뉴 알림을 설정할 수 있게 한다.

이 파일은 앞으로 추가할 기능을 정리하는 작업용 기능 지도이다.

## 내비게이션

* 홈

    * 현재 활성화된 학교의 급식표를 보여준다.
    * 학교가 있는 요일인 월요일, 화요일, 수요일, 목요일, 금요일만 보여준다.
    * 앱이 아직 설정되지 않은 경우, 급식표 대신 초기 설정 폼을 보여준다.
* 설정

    * NEIS API 키를 저장한다.
    * 학교를 검색하고 저장한다.
    * 선호 메뉴 조식 알림 시간을 선택한다.

## 초기 설정

앱은 다음 두 조건이 모두 충족되었을 때 설정이 완료된 것으로 본다.

* NEIS API 키가 있다.
* 학교가 선택되어 있다.

NEIS API 키는 다음 경로에서 가져올 수 있다.

* 사용자가 입력하여 앱 환경설정에 저장한 값
* `local.properties`의 예비 값. `BuildConfig.NEIS_API_KEY`를 통해 사용한다.

학교 정보는 다음 경로에서 가져올 수 있다.

* NEIS `schoolInfo`에서 사용자가 선택한 학교
* `local.properties`의 예비 값. `NEIS_OFFICE_CODE`, `NEIS_SCHOOL_CODE`, `NEIS_SCHOOL_NAME`을 통해 사용한다.

## 급식 불러오기

* 앱은 한 번에 한 학교 주간 급식표를 불러온다.
* 월요일부터 금요일까지는 현재 주의 월요일을 활성 주간의 시작일로 본다.
* 토요일이나 일요일에는 다음 주 월요일을 활성 주간의 시작일로 본다.
* 앱은 NEIS에 다음 값을 사용하여 한 주 전체 범위를 요청한다.

    * `MLSV_FROM_YMD`
    * `MLSV_TO_YMD`
* 급식 정보는 Room에 다음 기준으로 캐시된다.

    * 시도교육청 코드
    * 학교 코드
    * 급식 일자
    * 급식 코드

## 날짜 선택

* 홈 화면에는 월요일부터 금요일까지 5개의 날짜 버튼이 한 줄로 표시된다.
* 선택된 날짜에 따라 표시할 캐시된 급식 정보가 결정된다.
* 주말 급식은 UI에 노출하지 않는다.

## 학교 검색

* 학교 검색은 NEIS `schoolInfo`를 사용한다.
* 사용자는 최소 두 글자 이상 입력해야 한다.
* 검색 결과에는 가능한 경우 학교명, 교육청, 학교 종류, 주소를 표시한다.
* 학교를 선택하면 다음 정보가 저장된다.

    * 시도교육청 코드
    * 학교 코드
    * 학교명
* 학교를 선택한 뒤에는 활성 주간의 급식표를 새로고침한다.

## 선호 메뉴

* 선호 메뉴는 DataStore 환경설정에 문자열 집합으로 저장된다.
* 선호 메뉴는 급식 메뉴 행에서 바로 토글할 수 있다.
* 각 메뉴 행에는 선호 메뉴 동작이 있다.

    * `선호`는 해당 메뉴를 선호 키워드로 저장한다.
    * `해제`는 일치하는 선호 키워드를 제거한다.
* 일치 여부를 판단할 때는 공백과 대소문자를 무시한다.
* 일치 방식은 부분 문자열 기반이다.

    * 선호 메뉴 `돈까스`는 `수제돈까스`와 일치한다.
* 선택된 날짜의 급식에 선호 메뉴가 포함되어 있으면 다음과 같이 표시한다.

    * 급식 목록 위에 배너를 보여준다.
    * 일치하는 메뉴 행을 시각적으로 강조한다.
    * 선호 메뉴가 포함된 요일 버튼에는 별도의 색상을 적용한다.

## 선호 메뉴 조식 알림

* 앱은 사용자가 선택한 시간에 매일 로컬 알람을 예약한다.
* 기본 알림 시간은 11:40이다.
* 설정 탭은 알림 시각의 시와 분을 앱 환경설정에 저장한다.
* 저장된 시간이 변경되면 매일 알람을 다시 예약한다.
* 알람 시간이 되면 앱은 캐시된 급식 정보에서 다음 날 조식을 확인한다.
* 다음 날 조식에 선호 메뉴가 포함되어 있으면 알림을 보낸다.
* 알림 문구 형식은 다음과 같다.

    * `다음날 조식에 돈까스가 포함돼 있어요.`
    * `다음날 조식에 밥이 포함돼 있어요.`
* `이/가` 조사는 마지막 한글 음절의 받침 유무에 따라 선택한다.
* Android 13 이상에서는 알림 권한이 필요하다.
* 기기가 부팅된 뒤에는 알람을 다시 예약한다.

## 현재 저장 구조

* DataStore 환경설정:

    * NEIS API 키
    * 선택된 학교
    * 선호 메뉴 키워드
* Room:

    * 캐시된 급식 행

## 알려진 후속 아이디어

* 실제 배포용으로는 NEIS API 키 저장 방식을 암호화 저장소로 옮긴다.
* 이전 주/다음 주 이동 기능을 추가한다.
* 선호 메뉴 일치 알림 기능을 추가한다.
* 주간 시작일 계산과 선호 메뉴 일치 로직에 대한 테스트를 추가한다.
* 더 보기 좋은 아이콘 기반 하단 내비게이션을 추가한다.
